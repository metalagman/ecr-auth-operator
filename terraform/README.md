# Terraform

This directory contains Terraform code for installing and operating `ecr-auth-operator`.

- Reusable module: `modules/ecr-auth-operator`
- GitHub module source: `git::https://github.com/metalagman/ecr-auth-operator.git//terraform/modules/ecr-auth-operator?ref=v0.0.2`
- Runnable example: `examples/ecr-auth-operator`

Useful commands from repository root:

```bash
task terraform:fmt
task terraform:validate
```
